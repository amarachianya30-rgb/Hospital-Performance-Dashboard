# Hospital-Performance-Dashboard

Interactive Power BI dashboard analyzing hospital performance, patient metrics and operational efficiency.

### Project Overview

This project presents an interactive Hospital Performance Dashboard developed using Power BI. The dashboard provides insights into hospital operations, patient activity and key performance indicators to support data-driven decision-making.

### Objectives

- Monitor hospital performance metrics
- Analyze patient trends and activity
- Track operational efficiency
- Identify areas for performance improvement

### Tools Used

- Power BI
- Microsoft Excel
- Data Visualization
- Data Analysis

### Key Features

- Patient Metrics Dashboard
- Performance KPI Tracking
- Trend Analysis
- Interactive Visualizations
- Executive Summary Reporting
